## 目标

让 Claude Code 在终端中执行命令，理解授权和确认机制。

## 任务

Claude Code 可以直接在终端中运行命令。每次执行前会展示命令内容，需要你确认批准。

1. 请 Claude Code 运行命令查看当前目录的内容：`ls -la`（Windows 上可以用 `dir`）
2. 观察命令执行前出现的确认提示
3. 批准执行，查看输出结果
4. 再试试让 Claude Code 运行 `pwd`（或 Windows 上的 `cd`）查看当前路径

完成后告诉我：运行命令时看到了什么确认界面？你觉得这个安全机制有什么用？

## 验证

告诉我你运行了什么命令，以及看到了什么输出。我会评估你理解了命令执行的流程和安全机制。

## 提示

- Claude Code 执行命令前会显示命令内容和"批准/拒绝"选项
- 这个安全机制可以防止误操作
- 你可以在设置中将常用命令加入白名单，减少确认提示
- 试试说："帮我运行 ls -la 看看当前目录有什么"
