## 目标

让 Claude Code 帮你读写文件。这是日常使用中最常见的操作之一。

## 任务

1. 请 Claude Code 在当前目录创建一个名为 `hello-quest.txt` 的文件，内容为 "Hello, Claude Code Quest!"
2. 然后请它读取这个文件，确认内容正确
3. 最后请它把文件内容改为 "Level 3 Complete!"

完成后告诉我你已经完成了。

## 验证

我会检查 `hello-quest.txt` 是否存在，以及内容是否为 "Level 3 Complete!"。

## 提示

可以这样对 Claude 说：
- "帮我创建一个 hello-quest.txt 文件，内容是 Hello, Claude Code Quest!"
- "读取 hello-quest.txt 看看内容对不对"
- "把 hello-quest.txt 的内容改成 Level 3 Complete!"
