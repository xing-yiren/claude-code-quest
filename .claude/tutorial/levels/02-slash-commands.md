## 目标

掌握 Claude Code 的斜杠命令，这是最常用的操作方式。

## 任务

Claude Code 提供了大量以 `/` 开头的斜杠命令。

1. 输入 `/help` 查看所有可用命令列表
2. 浏览命令列表，找到至少 3 个你感兴趣的或觉得有用的命令
3. 输入 `/clear` 清除当前对话历史（对话太长时这个命令很常用）

完成后告诉我：你看到了哪些类别的命令？你最感兴趣的是哪 3 个？

## 验证

告诉我你看到的 3 个斜杠命令及其作用。我会根据 Claude Code 实际的命令列表来评估。

## 提示

- 在输入框中输入 `/` 会自动弹出命令补全列表
- `/help` 会展示所有命令及简短说明
- `/clear` 可以清除对话历史，释放上下文空间
- 试试 `/model` 看看能切换哪些模型
