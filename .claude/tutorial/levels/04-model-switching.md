## 目标

学习如何切换 Claude 模型，了解不同模型的特点和适用场景。

## 任务

Claude Code 支持多个模型。不同模型在速度、能力和成本上各有侧重。

1. 输入 `/model` 查看当前可用的模型列表
2. 观察有哪些模型可供选择
3. 尝试切换到一个不同的模型（可以切回来）

完成后告诉我：你看到了哪些模型选项？你觉得什么场景下应该用 Opus，什么场景下用 Sonnet 或 Haiku？

## 验证

告诉我 Claude Code 支持哪些模型，以及它们各自适合什么场景。

## 提示

- `/model` 命令会显示可用模型列表，用方向键选择
- Opus：最强能力，适合复杂推理任务
- Sonnet：速度和能力的平衡，日常使用首选
- Haiku：最快速度，适合简单任务
