## 目标

了解 Claude Code 的对话界面，完成第一次交互。

## 任务

你现在就在 Claude Code 中！观察一下界面：
- 上方是对话历史区，显示你和 Claude 的对话
- 底部是输入框，你可以在这里输入消息
- 当你发送消息后，Claude 会回复你

请发送一条消息给 Claude Code，内容不限。例如：
- "你好，帮我介绍一下这个项目"
- "这个目录里有哪些文件？"
- 任何你想说的

发送后告诉我你已经完成了。

## 验证

你确认已发送消息即通过。这是教程的起点，后面的关卡会越来越有趣。

## 提示

在底部的输入框中输入文字，然后按 Enter 发送。
