## 目标

了解如何配置和使用自定义模型，让你可以用自己的 API key 接入更多模型选择。

## 任务

Claude Code 除了内置模型，还支持配置自定义模型（Custom Models）。这意味着你可以接入自己的 API key，使用不同的模型提供商。

1. 输入 `/model` 打开模型选择界面，看看有没有自定义模型相关的选项
2. 了解配置自定义模型需要哪些信息（提示：通常需要提供商、模型名称、API key）
3. 如果你有自己的 API key，可以尝试添加一个自定义模型；如果没有，了解一下配置方式也可以

完成后告诉我：配置自定义模型需要哪几样东西？使用自定义模型有什么好处和需要注意的地方？

## 验证

告诉我配置自定义模型需要的信息，以及你觉得什么场景下会用到它。

## 提示

- 自定义模型通过设置文件（settings.json）或 `/model` 界面配置
- 不同提供商（如 Anthropic、OpenAI）的配置格式可能不同
- 使用自定义模型时，API 调用费用由你的 API key 承担
- 如果不想实际配置，光了解概念也可以过关
