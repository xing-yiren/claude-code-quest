## 目标

理解 CLAUDE.md 的作用，学会用 `/init` 生成项目指南。

## 任务

CLAUDE.md 是 Claude Code 的项目指南文件。它告诉 Claude 你的项目结构、代码规范和使用习惯，让 Claude 更好地为你工作。

1. 输入 `/init` 命令
2. Claude Code 会自动分析当前项目，生成一份 CLAUDE.md
3. 阅读生成的 CLAUDE.md，理解它的结构

完成后告诉我：CLAUDE.md 包含哪些部分？你觉得它对日常使用有什么帮助？

## 验证

我会检查 CLAUDE.md 文件是否存在且内容不为空。

## 提示

- `/init` 会扫描项目文件结构，自动生成描述
- 你可以手动编辑 CLAUDE.md 来补充更多信息
- 好的 CLAUDE.md 可以显著提升 Claude Code 的回答质量
- 试试对 Claude 说："根据 CLAUDE.md 了解这个项目"
