## 目标

理解 Claude Code 中会话的继续与回退机制，学会在中断后恢复工作，以及在走错方向时回到对话的较早节点重新尝试。

## 概念

### 什么是 `/resume`？

`/resume` 用于恢复之前的会话。当你关闭了 Claude Code、切换了终端，或者想继续昨天的工作时，可以用它回到旧 session，接着上次的上下文继续。

### 什么是 rewind？

rewind 用于把当前对话回退到之前的某个节点。适合在你发现刚才的思路不对、想撤销后续对话并重新尝试时使用。

### 它们分别适合什么场景？

- **工作中断后继续** — 用 `/resume`
- **试错后回到 earlier node** — 用 rewind
- **比较不同方案** — 回退后重新尝试另一条路径

## 任务

1. 输入 `/resume`，观察 Claude Code 如何展示可恢复的会话
2. 恢复一个之前的会话，看看是否能接着聊
3. 回到当前工作后，再尝试一次 rewind，观察它回退到了哪里
4. 对比一下：`/resume` 和 rewind 各适合什么场景

## 提示

- `/resume` 更像"回到之前的会话"
- rewind 更像"回到当前会话里较早的一个节点"
- 一个管恢复，一个管回退
